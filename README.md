# สบายมาร์เก็ต

เว็บขายสินค้าสแตติกสำหรับเปิดใช้งานทันทีในเบราว์เซอร์ มีสินค้า mock, ค้นหา, กรองหมวด, ตะกร้า, โค้ดส่วนลด และฟอร์มสรุปคำสั่งซื้อ

## เปิดใช้งาน

เปิดไฟล์ `index.html` ได้โดยตรง หรือใช้ live server ใดก็ได้ถ้าต้องการพัฒนาเพิ่มเติม

## ไฟล์หลัก

- `index.html` โครงหน้าเว็บ
- `styles.css` งานออกแบบ responsive
- `script.js` ข้อมูลสินค้า ตะกร้า และ checkout mock
- `assets/hero-products.png` ภาพ hero สร้างด้วย image generation
