# สบายมาร์เก็ต Premium Tech

เว็บขายสินค้าแบบ static multi-page สำหรับร้านแกดเจ็ตและสินค้า premium tech มีหน้าแยกชัดเจนสำหรับหน้าแรก สินค้า ดีลพิเศษ ตะกร้า Checkout และบริการ

## เปิดใช้งาน

เปิดไฟล์ `index.html` ได้โดยตรง หรือใช้ live server ใดก็ได้ถ้าต้องการพัฒนาเพิ่มเติม

## หน้าเว็บ

- `index.html` หน้าแรกพร้อม hero, หมวดสินค้า, featured products
- `products.html` ค้นหา กรองหมวด และเรียงสินค้า
- `deals.html` ดีล bundle และโปรโมชัน
- `cart.html` ตรวจรายการ ปรับจำนวน และใช้โค้ดส่วนลด
- `checkout.html` กรอกข้อมูลจัดส่งและสรุปคำสั่งซื้อ
- `support.html` รายละเอียดบริการ จัดส่ง รับประกัน และ FAQ

## ไฟล์หลัก

- `styles.css` งานออกแบบ responsive, motion, hover effects และ page layout
- `script.js` ข้อมูลสินค้า ตะกร้าข้ามหน้า ดีล bundle โค้ดส่วนลด และ checkout mock
- `assets/hero-products.png` ภาพ hero สำหรับหน้าแรก
